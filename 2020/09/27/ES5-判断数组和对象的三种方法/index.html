判断数组和对象的三种方法
发表于2020-09-27|更新于2020-11-15|ES5
|阅读量:

一、方法

1. toString

可以具体判断es5数据类型

  • Object.prototype.toString.call({}) –> [object Object]
  • Object.prototype.toString.call([]) –> [object Array]
  • Object.prototype.toString.call(123) –> [object Number]
  • Object.prototype.toString.call([]) –> [object Array]

2. constructor

  • [].constructor –> function Array(){}
  • var obj = {}; obj.constructor –> function Object(){}

3. instanceof

instanceof 判断Array.prototype是否出现在[],{}的原型链上

  • [] instanceof Array –> true
  • {} instanceof Array –> false

二、深度克隆-toString

  1. 先声明三个变量target toStr arrStr
  2. 对原始对象origin进行遍历
  3. 判断prop是不是origin自身属性值(非原型链属性)
  4. 判断prop是原始值还是引用值
  5. 如果是原始值则继续判断，如果是引用值则直接克隆
  6. 判断prop是数组还是对象
  7. 如果是数组，那么给target的prop属性赋值为空数组[]，否则赋值为空对象{}
  8. 递归判断
