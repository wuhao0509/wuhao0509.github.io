一、对象解构

1.1 ES5从对象中取出属性值的方法

  1. 下面这样取值太麻烦，所以ES6给出了解构

1.2 什么是解构

  1. 使用ES6的一种语法规则，将一个对象或数组的某个属性提取到已经声明的同名变量中

  2. 解构不会对被解构的目标造成任何影响

1.3 解构写法

  1. 写法
  1. 语法糖：let {name,age} = obj1;

1.4 在解构中使用默认值

  1. {同名变量 = 默认值}：let {name,age=100} = obj1;

1.5 非同名属性解构

  1. {属性名:变量名}：let {name,age,sex:jender} = obj1;

1.6 深层次解构

  1. 大括号内放的始终是新变量：let {name,age,adress:{provience}} = obj1;

二、数组解构

const numbers = [‘a’,’b’,’c’,’d’];

  1. 按照对象的方式解构数组：const { 0:n1 , 1:n2 } = numbers

  2. 正常写法

  • let n1,n2;([n1,n2] = numbers)

  • 语法糖：let [n1,n2] = numbers

  1. 取数组第一项和最后一项: const [n1,,,n4] = numbers
  1. 数组嵌套数组解构
  • 得到数组下标为4的数组中下标为2的数据，存放到变量n中，const [,,,,,[,,n]] = numbers
  1. 数组嵌套对象解构
  • 解构属性a的值：const [,,,,{a}] = numbers;
  1. 对于剩余项的解构

  • 对象：const { name, ...obj } = user;

  • 数组: const [a, b, ...nums] = numbers;

  1. 用数组解构交换两个变量

  • 以前借助第三个变量

  • [b,a] = [a,b]

三、参数解构(用的最多)

  1. 如果实参传入的是对象或数组，那么可以在形参的位置直接解构，一般会给默认值

  2. 如果实参是undefined、null的解构，会报错

  • 解决办法：直接给形参整体一个默认值
