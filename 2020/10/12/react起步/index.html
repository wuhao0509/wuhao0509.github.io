一、创建并渲染React元素的两种方法

1.1 直接在页面上使用React准备工作

  1. crossorigin：如果是跨域资源报错不会显示详细信息，加上crossorigin就可以显示详细信息了

1.2 js语法创建

1.2.1 React.createElement

  1. 创建一个React元素，本质上是一个对象，所以称作虚拟DOM，依赖于1.1中的第一个文件
  • 参数1：元素类型，如果是字符串，一个普通的HTML元素
  • 参数2：元素的属性，一个对象
  • 后续参数：元素的子节点

1.2.2 ReactDOM.render

  1. 将虚拟dom转换成真正的dom，并将dom节点放到页面的容器内，依赖于1.1中的第二个文件
  • 参数1：React.createElement创建的元素
  • 参数2：页面容器

1.3 JSX

  1. JS的扩展语法，将创建虚拟dom的方法React.createElement方法简化，依赖于1.1中的第三个文件

  2. ES标准不支持JSX语法，所以需要使用babel进行转义

二、使用脚手架搭建工程

2.1 脚手架种类

  1. 官方：create-react-app，第三方：next.js、umijs

2.2 安装脚手架步骤

  1. yarn create create-react-app:yarn create 指令，全局安装并马上运行脚手架工具

  2. yarn start:运行工程，package.json脚本里有

  3. yarn build:部署(打包)工程，package.json脚本里有
