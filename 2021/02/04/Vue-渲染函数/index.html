一、基础

  1. 当我们需要使用JavaScript的编程能力时，可以利用渲染函数。渲染函数比模板更接近于编译器。

  2. render函数跟template一样都是创建 html 模板的，但是有些场景中用 template 实现起来代码冗长繁琐而且有大量重复，这时候就可以用 render 函数。

  3. 例如，我们想要生成一些标题，如果，我们按照之前的方式，那么模板内将会十分冗余，如果此时利用渲染函数，那么代码写起来将会简洁很多。
