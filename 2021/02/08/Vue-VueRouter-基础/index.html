一、什么是路由？

  1. 路由是根据不同的url地址展现不同的内容或页面，早期的路由都是后端直接根据url来重载页面实现的，即后端控制路由。

  2. 后来页面越来越复杂，服务器压力越来越大，随着ajax（异步刷新技术）的出现，页面的实现非重载就能刷新数据，让前端也可以控制url自行管理，前端路由由此而生。

二、什么时候使用前端路由？

  1. 前端路由更多用在单页应用上，也就是SPA(Single Page Web Application)，在单页面应用中，大部分页面结果不变，只改变部分内容的使用。

六、hash模式

  1. vue-router 默认 hash 模式 —— 使用 URL 的 hash 来模拟一个完整的 URL，于是当 URL 改变时，页面不会重新加载。
